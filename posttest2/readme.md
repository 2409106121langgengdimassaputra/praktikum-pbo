Deskripsi Program

Program ini merupakan aplikasi sederhana yang dibuat menggunakan bahasa pemrograman JavaScript untuk mengelola data penjualan UC pada permainan PUBG Mobile. UC merupakan mata uang virtual yang digunakan dalam game PUBG Mobile untuk membeli berbagai item seperti pakaian karakter, skin senjata, dan royal pass. Program ini dijalankan melalui terminal dan memiliki menu interaktif yang dapat digunakan oleh pengguna untuk mengelola data paket UC.

Pada program ini pengguna dapat melakukan pengolahan data paket UC dengan beberapa fitur utama yaitu menambah data, melihat data, memperbarui data, dan menghapus data. Program akan terus menampilkan menu utama dan berjalan secara berulang sampai pengguna memilih menu keluar dari program.

Tujuan Program

Tujuan dari pembuatan program ini adalah untuk menerapkan konsep pemrograman berorientasi objek dalam JavaScript khususnya konsep encapsulation. Selain itu program ini juga bertujuan untuk memahami penggunaan class sebagai pembentuk objek data, penggunaan array sebagai tempat penyimpanan objek, serta penerapan operasi CRUD yang terdiri dari create, read, update, dan delete.

Konsep Encapsulation

Encapsulation merupakan konsep dalam pemrograman berorientasi objek yang digunakan untuk menyembunyikan data dari akses langsung di luar class. Data dalam class dibuat bersifat private sehingga tidak dapat diakses secara langsung dari luar class. Untuk mengakses atau mengubah data tersebut diperlukan method khusus yang disebut getter dan setter.

Dalam program ini encapsulation diterapkan pada class UC dengan menjadikan atribut id, namaPaket, jumlahUC, dan harga sebagai private property. Atribut tersebut hanya dapat diakses melalui method getter dan dapat diubah menggunakan method setter yang telah disediakan dalam class.

Struktur Data

Data yang dikelola dalam program ini adalah data paket UC PUBG Mobile. Setiap paket UC memiliki beberapa informasi yaitu id paket, nama paket, jumlah UC yang tersedia, dan harga paket tersebut. Data ini disimpan dalam bentuk objek yang dibuat dari class UC.

Penyimpanan Data

Seluruh data paket UC disimpan dalam sebuah array JavaScript. Array ini berfungsi sebagai tempat penyimpanan objek yang dibuat dari class UC. Ketika pengguna menambahkan paket UC baru maka objek tersebut akan dimasukkan ke dalam array sehingga dapat ditampilkan, diperbarui, atau dihapus saat diperlukan.

Fitur Program

Program memiliki beberapa fitur utama yang dapat digunakan oleh pengguna. Fitur pertama adalah menambah data paket UC. Pada fitur ini pengguna diminta memasukkan informasi paket seperti id paket, nama paket, jumlah UC, dan harga. Data yang dimasukkan akan disimpan ke dalam array.

Fitur kedua adalah melihat data paket UC. Pada fitur ini program akan menampilkan seluruh data paket UC yang tersimpan dalam sistem sehingga pengguna dapat melihat daftar paket yang tersedia.

Fitur ketiga adalah memperbarui data paket UC. Pada fitur ini pengguna dapat mengubah data paket UC yang sudah ada dengan cara memasukkan id paket yang ingin diperbarui kemudian memasukkan data baru untuk menggantikan data sebelumnya.

Fitur keempat adalah menghapus data paket UC. Pengguna dapat menghapus paket UC dari sistem dengan memasukkan id paket yang ingin dihapus dari daftar data.

Fitur terakhir adalah keluar dari program. Jika pengguna memilih menu keluar maka program akan berhenti dijalankan.

Alur Program

Saat program dijalankan sistem akan menampilkan menu utama kepada pengguna. Pengguna kemudian memilih salah satu menu yang tersedia seperti menambah data, melihat data, memperbarui data, atau menghapus data. Program akan menjalankan fungsi sesuai dengan pilihan pengguna. Setelah proses selesai dilakukan maka program akan kembali menampilkan menu utama. Proses ini akan terus berulang sampai pengguna memilih menu keluar sehingga program akan berhenti dijalankan.

Kesimpulan

Program Sistem Penjualan UC PUBG Mobile ini dibuat untuk mengelola data paket UC secara sederhana menggunakan JavaScript. Program ini menerapkan konsep pemrograman berorientasi objek khususnya encapsulation dengan menggunakan private property serta method getter dan setter. Selain itu program ini juga menerapkan operasi CRUD untuk mengelola data serta menggunakan array sebagai tempat penyimpanan objek. Dengan adanya program ini pengolahan data paket UC dapat dilakukan dengan lebih terstruktur melalui terminal.
